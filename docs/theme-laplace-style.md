# 主题升级说明（Laplace 风格方向）

## 目标
- 让博客从“演示站”升级为“可长期运营的个人站”视觉质感。
- 保持可读性优先，避免过度动画影响阅读。

## 已实施内容
- 首页改为叙事式布局：主内容区 + 侧栏信息面板。
- 导航、卡片、Hero 区统一为半透明面板与冷色发光体系。
- 阅读页增加独立头部信息区与正文容器。
- Markdown 样式系统化：标题、引用、表格、行内代码、代码块复制按钮统一。
- 全局字体从默认系统栈切换为更有识别度的组合（标题/正文分离）。

## 后续建议
- 增加文章标签色系与封面图占位，提升列表辨识度。
- 增加阅读页目录（TOC）与阅读进度条，优化长文体验。
